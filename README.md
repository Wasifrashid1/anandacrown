# Ananda Crown Mohali.

This repository contains the complete source code for the Mohali Next web project.
It is a fully independent project with no third-party platform dependency.

## Project Overview
The project is built using modern frontend technologies and is suitable for production deployment on any standard hosting platform.

## Technologies Used
- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn-ui

## Local Development Setup.

### Requirements
- Node.js (v18 or later)
- npm

### Installation Steps
```bash
git clone <REPOSITORY_URL>
cd <PROJECT_NAME>
npm install
npm run dev
